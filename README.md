# Grok Imagine

Grok Imagine is always making changes, use with caution

## Overview
Grok Imagine is a versatile tool designed to manage your favorite media. With its user-friendly interface, you can easily sort, categorize, and retrieve your favorite images and videos.

## Features
- **Image Management:** Sort and categorize images.
- **Video Management:** Keep track of your videos effortlessly.

## Important Notes
- download (all and video) and upscale functions only affect most recently generated video
- Make sure to save your changes regularly.
